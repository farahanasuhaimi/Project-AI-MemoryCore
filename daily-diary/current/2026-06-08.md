## Projek Baru: amin-maju

Buat sistem untuk bisnes kontraktor kakak — dia bina rumah dari beli tanah sampai serah kunci. Tiga pengguna: Kak Amin (owner), suami dia (worker), dan Nufa sendiri (bookkeeper).

**Yang dibina hari ni:**
- Laravel 12 + Tailwind v4 + Alpine.js
- 9 fasa pembinaan auto-seeded bila buat projek baru (dalam BM)
- Task checklist dengan 3 jenis: general, sebut harga, info-gathering
- Sebut harga keluar + masuk dengan PDF print
- Timeline hubungan (info contacts)
- Bookkeeping: perbelanjaan, bayaran klien, gaji pekerja
- Laporan P&L — per projek dan overview keseluruhan
- Role-based access — owner nampak semua, worker nampak task je, nufa nampak finance

**Bug yang ditemui dan dibaiki:**
- Form nested dalam form → projek terarkib bila edit (fixed: pisahkan form)
- Dashboard hardcoded zeroes → sambung ke DB live
- Task checkbox buat full page reload → tukar ke AJAX fetch, toggle terus
- `@json()` dalam HTML attribute pecah bila ada koma dalam expression → pindah ke `<script>` tag
- Migration order salah (quotation_items sebelum quotations) → betulkan timestamp

## Key Insights

- Alpine.js `get itemKind()` computed from a `__rcCategories` JSON inject is a clean pattern for form field toggling without duplicating blade templates. No extra page, no extra controller — just one form that adapts.
- The enum-to-varchar migration problem is a good reminder: always use varchar for domain fields that might expand (like `rate_type`). Enums feel tidy but create migration friction every time scope grows.
- `sectionToCategory()` with `str_contains()` is pragmatic but fragile — if a section label deviates from convention (e.g. "DIRECT MANHOURS" instead of "DIRECT MANPOWER"), it falls to 'other'. Worth documenting as a known limitation.
- The 405 bug on PATCH forms is a recurring Laravel gotcha. Worth adding a note in code review to always check `@method()` is present when using non-GET/POST routes in forms.

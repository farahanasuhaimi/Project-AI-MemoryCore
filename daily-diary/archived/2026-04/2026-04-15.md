2. **Option 1 implemented** — Removed `/public/build` from `.gitignore`. From now on, `npm run build` locally before every push, and the compiled assets travel with the repo. Hostinger just serves the pre-built files.

3. **AI content generation for Reach Angles** — Nufa wanted to integrate LLM API to generate Takaful content variations from her reach angles. Built the full feature:

   - `AngleContentService` — DeepSeek API call (OpenAI-compatible), cooldown guard, auto-cleanup (keeps last 5 batches)
   - Settings → API Settings page with masked key field
   - ✨ button on each angle card → Alpine modal → 3 style cards (💬 Casual / 📖 Story / 📊 Factual) with 📌 pin
   - Content Library page — all pinned content browseable by style, with copy-to-clipboard

4. **SSL cert fix** — `cURL error 60` on Windows dev machine. Fixed with `->withoutVerifying()` on the HTTP client. No impact on Hostinger (Linux has valid CA bundle).

5. **Live migration** — Tables didn't exist on Hostinger. Used a temporary route `/run-migrations-temp-x9k2` to run `artisan migrate --force` remotely. Removed immediately after.

6. **Content Library mobile optimised** — Filter bar made horizontally scrollable, single-column cards on mobile, bigger tap targets for copy/unpin buttons.

**Project**: cms-takaful

### Context

Nufa wanted to evolve the Strategy Marketplace from a simple "sell your AI content" model into a proper strategy system with categories, filters, flows, and three creation modes.

### What Was Built

**18 files — new and modified.**

**Data layer:**
- `strategies` table — category, channel, audience, difficulty, type (script/flow), source (provided/ai_guided/self_made), content, status
- `strategy_steps` table — ordered steps for flow-type strategies, each with script, timing_note, branch_yes, branch_no
- `marketplace_listings` altered — `angle_content_id` made nullable, `strategy_id` FK added
- `marketplace_purchases` altered — `imported_strategy_id` added for strategy purchase copies

**Logic layer:**
- `Strategy` model with label helpers (`categoryLabel`, `channelLabel`, `audienceLabel`)
- `StrategyStep` model
- `StrategyAiService` — DeepSeek API generates script or full flow (3–6 steps with branch logic) based on category/channel/audience/difficulty + optional brief
- `StrategyController` — full CRUD, AI generate endpoint (JSON), save-generated (with steps), step add/edit/delete
- `MarketplaceStrategyController` — rewritten to support both old angle_content listings and new strategy listings, with filters and proper purchase copy (strategy + all steps duplicated to buyer)

**UI layer:**
- Strategy Library index — filterable card grid (category, channel, audience, difficulty, type, source)
- Create page — tab switcher: Self Made vs AI Guided; AI Guided calls generate endpoint live, previews result, submits as form
- Show page — script view, flow step cards with branch indicators, inline step edit forms, add step panel, marketplace list form
- Edit page — all fields editable
- Marketplace index — updated with filter bar + strategy meta badges
- Sidebar — "Strategy Library" link added

**Strategy creation modes:**
- Provided: platform-seeded, null user_id, visible to all
- AI Guided: DeepSeek writes the script or flow steps
- Self Made: blank form, consultant writes their own
